开始上手

Create React App#

LaboUI 是 React 的自定义组件库，所以你需要在 React 工程中安装并使用 LaboUI。

npx create-react-app my-app

npx 是 npm 5.2 版本之后支持的 package 运行工具，请参阅 npm 相关文档的介绍。

tip

推荐使用 Create React App创建 React 工程。

有很多种方式安装 Create React App，具体参见官方文档。

安装#

使用 npm 安装 LaboUI 。执行如下命令：

npm install laboui

styled-components 的工具安装（可选）#

babel-plugin-styled-components

createGlobalStyle
